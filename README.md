### how install in LINUX?
```
$ git clone https://github.com/MohmmadALbaqer/install-scrcpy.git
$ cd install-scrcpy
$ pip install -r requirements.txt
$ chmod +x *
$ sudo bash install-scrcpy
```

### How it works?
```
$ python3 scrcpy.py
```
